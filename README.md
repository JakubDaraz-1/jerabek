# Komunitní Kalendář

Tento projekt je webová aplikace pro správu komunitních akcí a sdílených kalendářů.  
Cílem je umožnit uživatelům:

- vidět hlavní (centrální) kalendář,
- přidávat se na akce,
- importovat vlastní kalendář pomocí ICS,
- spravovat osobní kalendář,
- získávat kombinovaný kalendář ze všech zdrojů.

## Funkce aplikace
- Registrace a přihlášení uživatelů  
- Role (Admin, Organizátor, Uživatelský účet)  
- Centrální kalendář komunitních akcí  
- Přihlášení/odhlášení k událostem  
- Osobní kalendář  
- Import kalendáře přes ICS  
- Export do ICS

## Technologie navržene
- Backend: Python (FastAPI/Django)
- Frontend: React / Next.js + FullCalendar
- Databáze: PostgreSQL
